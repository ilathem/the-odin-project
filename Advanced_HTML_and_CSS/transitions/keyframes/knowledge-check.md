# Keyframes

1. What are the long and short-hand notations for CSS animations?

Long notations;
`animation-delay`
`animation-direction`
`animation-duration`
`animation-fill-mode`
`animation-iteration-count`
`animation-name`
`animation-play-state`
`animation-timeline`

Short Notation:
`animation: duration | easing-function | delay |
iteration-count | direction | fill-mode | play-state | name `
